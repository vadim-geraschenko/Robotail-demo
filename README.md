# Robotail ROI Card Demo

Интерактивное демо карточки товара для автомата морской бой «Бродяги морей».

Цель: показать, как карточка развлекательного оборудования может продавать через сценарий использования, коммерческие преимущества и расчёт окупаемости.

Это standalone frontend demo. Оно не является production-интеграцией и не заменяет текущий сайт.

## Stack

React, TypeScript, Vite, Tailwind CSS, Framer Motion.

## Run

```bash
npm install
npm run dev
```
